Smart Autonomous Crop Monitoring & Target Spraying UGV

94.5%Target Accuracy
4-LayerCustom PCB
30%Chemical Saved

Developed an AI-driven, terrain-adaptive Unmanned Ground Vehicle designed to traverse crop rows autonomously, detect disease patches using real-time Edge-AI processing, and execute localized target micro-spraying.


Abstract: Precision agriculture minimizes chemical surface runs while preserving overall environmental health. This project features a zero-turn mechanical chassis driven by high-torque brushed DC motors configured with close-loop encoder feedback loops via STM32 hardware timers.

A layered system architecture separates basic navigation telemetry from advanced workloads. A Jetson Nano edge processor hosts quantized MobileNetV2 pipelines, receiving digital video input from global-shutter camera feeds to locate crop anomalies. Detected segments feed micro-sprayer solenoids controlled by custom power-MOSFET driver topologies on a custom integrated motherboard. Results validate notable field reliability under volatile outdoor luminosity constraints.
